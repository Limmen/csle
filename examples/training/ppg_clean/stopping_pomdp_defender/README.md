# Phasic Policy Gradient (PPG) for POMDP

This directory contains example scripts for optimizing defender policies using PPG for the POMDP from [https://ieeexplore.ieee.org/document/9779345](https://ieeexplore.ieee.org/document/9779345)

## Commands

To run a script, execute:
```bash
python <script_name>
```
